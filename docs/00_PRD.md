# PRD: LifeCanvas

## 概要

LifeCanvasは、一般家庭向けのWindows用ライフプランシミュレーターである。

家計簿ではなく、住宅ローン・教育費・資産運用・税金・社会保険・年金・ライフイベントを統合し、将来40年間の資産・キャッシュフロー・老後資金を高精度にシミュレーションすることが目的。

## 対象ユーザー

- 一般ユーザー（FP知識不要）
- 初期は開発者自身が使用
- 設計は汎用性を重視

## 課題

- 既存のライフプランツールはWeb依存、または細かすぎて挫折する
- 住宅ローン・NISA・教育費・年金が連動するシミュレーターが存在しない
- シナリオ比較（金利変動・妻の働き方・子供の人数）が困難

## ゴール

| 優先度 | 項目 |
|--------|------|
| P0 | 40年間のキャッシュフロー・純資産シミュレーション |
| P0 | 住宅ローン（変動・固定・繰上返済・売却・賃貸） |
| P0 | 教育費テンプレート（保育園〜大学） |
| P0 | 投資（NISA・iDeCo・特定・現金） |
| P0 | 税金・社会保険の概算 |
| P1 | シナリオ比較（複数プラン並列） |
| P1 | PDF/Excelレポート出力 |
| P1 | AI分析（赤字検出・改善提案） |
| P2 | 給付金反映（児童手当・育休給付・ローン控除） |
| P2 | 車（購入・ローン・維持費・買替サイクル） |
| P2 | 保険（生命・医療・火災・自動車） |

## アーキテクチャ

**【Event-Driven Architecture】**
LifeCanvasのコアは「イベントドリブン設計」です。タイムライン上にイベント（育休、NISA変更、ローン返済など）を配置し、シミュレーションエンジンが時系列に沿って状態を変化させていくことで、複雑なライフプランを動的にシミュレーションします。（詳細は `04_EVENT_DRIVEN_ARCHITECTURE.md` 参照）

```text
┌─────────────────────────────────────────┐
│              PySide6 UI                 │
│  ┌──────────┐ ┌──────────┐ ┌─────────┐ │
│  │Timeline  │ │EventPanel│ │Report   │ │
│  └────┬─────┘ └────┬─────┘ └────┬────┘ │
│       │            │            │       │
│  ┌────▼────────────▼────────────▼────┐  │
│  │         ViewModel (MVVM)          │  │
│  └────────────────┬──────────────────┘  │
├───────────────────┼─────────────────────┤
│  ┌────────────────▼──────────────────┐  │
│  │   Event-Driven Simulation Engine  │  │
│  │  ┌────────┐ ┌────────┐ ┌───────┐  │  │
│  │  │Events  │ │State   │ │Engine │  │  │
│  │  └────────┘ └────────┘ └───────┘  │  │
│  └────────────────┬──────────────────┘  │
│  ┌────────────────▼──────────────────┐  │
│  │   Pydantic Models + SQLAlchemy    │  │
│  │           SQLite DB               │  │
│  └───────────────────────────────────┘  │
└─────────────────────────────────────────┘
```

## 技術スタック

| 用途 | 技術 | 理由 |
|------|------|------|
| UI | PySide6 | Windows ネイティブ、商用利用可 |
| データモデル | Pydantic | バリデーション・JSON変換が自動化 |
| ORM | SQLAlchemy | マイグレーション・リレーション管理 |
| DB | SQLite | ファイルベース、インストール不要 |
| グラフ | matplotlib | Python標準、PySide6埋め込み可 |
| PDF | ReportLab | 日本語フォント対応 |
| Excel | openpyxl | スタイル付きExcel出力 |

## 非ゴール（v1.0対象外）

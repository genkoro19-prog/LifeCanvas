# LifeCanvas

Windows向けライフプランシミュレーター。

## 方針

- 旧 `genkoro19-prog/FP` の計算ロジックとデータモデルを参考にする
- UIは全面的に再設計し、入力の迷いを減らす
- 計算根拠を年次・項目別に確認できる
- 生活費と住宅費、現金積立と消費支出などの二重計上を防止する
- NISA、住宅ローン、教育費、税金、社会保険、年金、ライフイベントを統合する
- 夫60歳定年、妻55歳退職、第二子の成長に連動した就労条件をサンプルとして搭載する

## 画面構成

1. かんたん設定
2. 将来結果ダッシュボード
3. 年表・年間内訳
4. シナリオ比較
5. 詳細設定

## 開発方針

- Python 3.12+
- PySide6
- Pydantic
- pytest
- JSON保存を先行し、SQLiteは後から追加

詳細は `docs/PRODUCT_SPEC.md` を参照してください。

# LifeCanvas

Windows向けライフプランシミュレーター。

家計簿ではなく、住宅ローン・教育費・働き方・NISA・年金・住み替えをまとめて比較し、将来の選択を支援します。

## 現在のMVP

- 結果ダッシュボード
- 重要項目だけの「かんたん設定」
- 年表と年間キャッシュフロー内訳
- 妻の働き方・金利・住み替えのシナリオ比較
- 夫60歳定年、妻55歳退職
- 第二子の成長に連動する妻のパート収入
- 住宅ローン金利1.68%から上限3%まで上昇
- NISA年間枠・1,800万円簿価枠・売却翌年の枠復活
- 現金不足時はNISA積立を先に減額し、それでも不足する場合だけ売却
- 生活費と住宅費の二重計上防止

## セットアップ

```powershell
py -3.12 -m venv .venv
.\.venv\Scripts\Activate.ps1
python -m pip install -e ".[dev]"
python main.py
```

## テスト

```powershell
pytest
```

1. 結果

## 注意

税金、社会保険、年金、住宅価格、育休給付などは将来予測を含む概算です。制度値と仮定を確認・編集できる設計へ段階的に拡張します。
